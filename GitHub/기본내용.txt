github 사용

많은 오픈 소스 프로젝트들이 사용함


각각의 개발자들이 진행한 개발의 변경 사항을 온라인에서 확인 가능한 서비스


Code Tab
소스코드를 바로 확인 가능

README.md
해당 git의 메뉴얼에 대한 내용이 적혀 있음

commits
하나의 Version 같은 것
각각의 commit은 해당 버전의 작성자와 변경사항 확인 가능

Repository
소스 코드의 변경 사항을 보관하는 장소
지역 저장소
- 개별 컴퓨터에 설치되어 있는 저장소
- 실질적인 업무가 진행됨
- 소스코드를 수정해 Commit 후 새로운 버전 형성
