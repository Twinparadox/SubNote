### 문서 객체 가져오기

## 하나의 문서 객체

# getElementById()
- 생성된 여러 문서 객체들 중, 하나의 문서 객체만 가져와 작업
- 매개변수로 HTML 요소의 id 사용
- HTML DOM에서 일반적으로 가장 많이 사용(객체 조작을 위해 기존 문서 객체를 가져와야 하기 때문)
- 입력 받은 id와 일치하는 요소 없으면 null 리턴


## 여러 개의 문서 객체

- 일치하는 여러 개의 문서 객체를 가져와 배열로 저장

# getElementsByName()
- 입력받은 문자열과 일치하는 name 속성을 가지는 문서 객체를 가져옴

# getElementsByTagName()
- 입력받은 태그 이름과 일치하는 문서 객체를 가져옴


* querySelector() : 선택자에 해당하는 첫 번쨰 한 개의 문서 객체를 가져오며 CSS 선택자임
* querySelectorAll() : 선택자에 해당하는 모든 문서 객체를 배열로 가져옴



### 문서 객체 제거

## removeChild()
- HTML 요소에 해당하는 문서 객체 제거
- 삭제하고자 하는 노드의 부모 노드로 이동해 부모 노드에서 자식 노드를 삭제함


## removeAttribute()
- HTML 요소의 속성이 객체화된 문서 객체를 제거
