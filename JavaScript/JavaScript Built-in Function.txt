### JavaScript 내장함수

- 자바스크립트에서 제공
- 구현하고자 하는 기능을 활용해 용이하게 활용 가능


* Infinity : 무한대를 나타내는 특수값
* NaN : Not a Number, 숫자 표현 불가능한 경우 입력되는 값


## alert()

- 사용자에게 메세지 전달을 위한 대화 상자 출력
- 확인만 선택 가능


## prompt()

- 사용자로부터 입력을 받기 위한 대화 상자 출력
- 확인, 취소 버튼 선택 가능
- 확인 : 입력받은 문자열 리턴, 취소 : null 리턴


## confirm()

- 사용자로부터 확인 받기 위한 대화 상자 출력
- 확인, 취소 버튼 선택 가능
- 확인 : ture, 취소 : false


## 타이머 함수

- 특정시간 후 혹은 간격으로 특정한 함수를 실행할 수 있게 하는 함수
- 함수와 시간을 매개변수로 이용
- ms 단위 사용
setTimeout(function, ms) : 일정 시간 경과 후, 함수 한 번 실행
clearTimeout(id) : 일정 시간 지난 후 함수 한 번 실행하는 것을 중지
setInterval(function, ms) : 일정 시간 간격으로 함수 반복 실행
clearInterval(id) : 일정 시간 간격으로 함수 반복해서 실행하는 것 중지


## 인코딩
- 정보의 형태나 형식을 변환하는 처리 방식
- 문자를 컴퓨터에 저장하거나 통신에 사용하기 위해 부호화하는 방법
- 웹 상에서 통신 시, 한글 등 유니코드의 오작동 방지 위해 사용
ex) 웹 검색 시 %, 알파벳 대문자 조합 주소

#escape()
- 영문 알파벳, 숫자, 일부 특수문자(@,*,-,_,+,.,/) 제외하고 모두 인코딩
- 1byte 문자는 %XX 형태로, 2byte는 %uXXXX 형태로 변환

#encodeURI()
- escape()함수에서 인터넷 주소에 사용되는 일부 특수 문자 (:,;,/,?,&)를 인코딩하지 않음
- 파라미터 전달하는 인터넷 주소 전체를 인코딩할 때 사용

#encodeURIComponent()
- 알파벳과 숫자를 제외한 모든 문자 인코딩


## 디코딩
- 인코딩 된 것을 다시 원래의 문자나 정보로 되돌림

#unescape()
- escape()를 이용해 인코딩한 정보를 다시 원래 정보로 디코딩

#decodeURI()
- encodeURI()를 이용해 인코딩한 정보를 원래 정보로 디코딩

#decodeURIComponet()
- encodeURIComponet()를 이용해 인코딩한 정보를 원래 정보로 디코딩


## eval()

- 문자열을 코드로 실행할 수 있음
- 표현식, 변수 등을 코드로 변환해 실행
- 함수 괄호 안의 문자열이 JavaScript 코드로 해석


## isFinite()

- 값의 유한성과 숫자 여부 판단 위해 사용
- +무한, -무한, NaN(Not a Number)의 경우 false, 나머지는 true


## isNaN()

- NaN 여부를 판단 위해 사용
- 값이 NaN이면 true, 아니면 false


## Number()

- 다른 자료형을 숫자로 변환할 때 사용
- 문자 포함 시, NaN 리턴


## parseInt()

- 문자열을 정수로 변환할 때 사용
- 문자열이 0x로 시작하면 16진수, 0으로 시작하면 8진수였으나 이는 현재 사용하지 않음
- 변환하고자 하는 문자열의 처음이 문자로 시작해 변환이 불가능하면 NaN 리턴
- 두 번째 매개변수에 진법을 입력 시, 앞의 수를 해당 진법의 수로 인식


## parseFloat()

- 문자열을 부동 소수점 수로 변환할 때 사용
- 문자열에서 숫자가 끝날 때까지 숫자로 변환
- 문자열의 처음이 문자로 시작해, 변환 불가능 시, NaN 리턴
