### JavaScript 상속

- 객체의 특성을 그대로 물려받는 또 다른 객체를 만들 수 있는 기능을 의미
ex. 부모 객체(부모 함수), 자식 객체(자식 함수)
- 기존의 특성을 수정하고 변경해 파생된 새로운 객체 만들 수 있음
- 객체에서 중요한 점은 계층보다는 동작
- 객체가 다른 객체로 바로 상속 가능
- JavaScript에서의 상속은 프로토타입을 이용해 구현

부모에 해당하는 함수를 통해 객체 생성 -> 자식에 해당하는 함수의 프로토타입 속성으로 생성한 객체 참조


## Object.create()

- 괄호 안에 객체를 지정해 새로운 객체를 생성하는 메서드
- 새로 생성된 객체는 지정된 객체의 프로토타입과 링크로 연결함
