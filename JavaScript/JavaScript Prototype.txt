### 프로토타입

- 객체 지향 언어 중 하나
- 객체의 자료 구조, 메서드를 동적으로 변경이 가능
- JavaScript
- 클래스 기반의 언어에 비해 정확성, 안정성, 예측 가능성이 떨어짐
- 속성과 메서드를 포함하는 객체
- 모든 객체는 자신의 프로토타입을 가리키는 prototype이라는 숨겨진 속성을 가짐
- 모든 객체는 속성을 상속하는 프로토타입 객체에 연결됨
- 표기법 {}를 이용해 생성된 객체의 경우 JavaScript의 표준 객체인 Object의 속성인 프로토타입(Object.prototype) 객체에 연결됨


## 프로토타입 링크

- 새로 생성된 모든 객체는 자신의 프로토타입 속성과 자신의 생성자 함수의 프로토타입 속성이 가리키는 프로토타입 객체와 링크로 연결되는 것
- 모든 객체에서 프로토타입 링크로 연결된 프로토타입 객체의 속성에 접근해 자신의 속성처럼 이용 가능


## 프로토타입 체인
- 상위의 프로토타입 객체들과 연결된 프로로타입 링크들의 집합


## 프로토타입 체이닝
- 프로토타입 체인을 따라 상위 프로토타입 객체를 순차 검색하는 것
