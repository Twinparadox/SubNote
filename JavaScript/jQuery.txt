### jQuery

- 브라우저에서 동작하는 Client Side JavaScript Library
- JavaScript 프로그래밍을 단순화 -> 간결한 코드로 많은 동작 구현
- 문서 객체 모델 조작, 이벤트 제어, 애니메이션, AJAX 개발
- 웹 표준만으로 플래시와, 실버라이트 사용과 유사한 효과 구현
- 모바일에서도 작동 가능
- 브라우저 간 차이로 발생하는 문제점 쉽게 해결 가능

* CDN(Content Delivery Network) : 콘텐츠를 효율적으로 전달하기 위해 여러 노드를 가진 네트워크에서 데이터를 저장하여 제공하는 방식
- 분산된 서버를 통해 jQuery를 사용할 수 잇게 하여 성능 향상
- 로딩 속도 향상(다운된 jQuery의 복사본이 있기 때문)
- CDN 호스트 : MS, Google
