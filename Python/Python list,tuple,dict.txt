리스트
- 임의의 객체를 순차적으로 저장하는 집합적 자료형
- 문자열이 지닌 대부분의 연산들을 리스트도 지원
- 객체를 순차적으로 저장하므로, 인덱스가 존재함
- 동일한 변수에 다른 리스트를 할당할 경우, 해당 변수의 레퍼런스 변경
- range(k) : 0부터 k-1까지의 숫자의 리스트 반환
- 4 in L : 멤버쉽 테스트, 리스트 L안에 4가 있는지 확인



튜플
- 리스트와 유사, 튜플 내 값은 변경 불능, 각 값에 대한 인덱스 부여
- 상수적인 성격을 갖고 있음(리스트는 내용 변경 가능)


사전
- 키를 이용해 값을 저장하는 자료 구조
- 저장된 각 자료에 대한 순서는 의미 없으나, 키에 대한 해시 값으로 순차 정렬
- key, value
- 새 항목 넣을 때는, dictionary['key']='value'
- d.keys() : key만 임의의 순서로 리스트로 추출
- d.values() : value만 임의의 순서로 리스트로 추출
- d.items() : key, value의 임의의 순서로 튜플로 추출 후 리스트로 반환


내장 자료형의 특성
- 수치형 : 값 변경 불가
- 문자열 : 시퀀스 자료형, 변경 불가
- 리스트 : 시퀀스 자료형, 변경 가능
- 튜플 : 시퀀스 자료형, 변경 불가
- 사전 : 매핑 유형, 변경 가능

type(A) : A의 자료형 확인
== : 자료형을 비교
id() : 객체의 식별자 반환
is 키워드 : 두 객체의 식별자 동일한지 체크
