함수

- 함수 이름에 저장된 레퍼런스를 다른 변수에 할당해 그 변수를 이용해서 함수 호출 가능
- 함수의 몸체에는 최소 1개 이상의 statement가 존재해야 함
- 아무 내용이 없는 몸체를 지닌 함수를 만들 때는 pass 예약어를 몸체에 적어줄 것
- 함수에서 다른 함수 호출 가능
- 함수 호출 시 인자의 이름과 함께 값을 넘겨줄 수 있음
- 인자의 default를 지정 가능
- 두 개 이상의 값을 동시 반환 가능
- 함수에 정의된 인자의 자료형은 호출시에 동적으로 결정됨
