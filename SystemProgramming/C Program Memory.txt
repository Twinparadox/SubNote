### C Program Memory

## 영역별 특성


# text
- 프로그램 코드 저장

# data
- 초기화 한 전역 변수, stack 변수 저장
- 프로그램 실행 전부터 존재, 유효 시간이 가장 긴 영역

# bss
- 초기화 하지 않은 전역 변수, static 변수 저장, 0으로 자동 초기화
- 같은 메모리에 저장되어, 전역 변수와 static 변수의 특성이 유사

# stack
- 지역변수와 함수의 파라미터, 리턴값 저장
- 자동 초기화되지 않음

# heap
- 변수명으로 접근 불가능
- 포인터나 주소로 접근
- 자료구조를 활용하여 사용하는 영역
