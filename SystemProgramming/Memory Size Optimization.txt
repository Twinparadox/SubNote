### Memory Size Optimization

## 워드 정렬
메모리 작업을 효과적으로 할 수 있게 데이터 타입에 따라 저장될 수 있는 주소 한정

## 패딩
워드 정렬 시, 데이터 사이에 생기는 빈 메모리 부분을 0으로 채움
패딩 비트로 인해서 메모리 낭비 발생 가능


멤버 변수 선언 순서에 따라서 패딩 비트 숫자 줄일 수 있음
선언 순서를 고려하여 메모리 최적화 가능

struct data
{
	char a;
	int b;
	short c;
	float d;
};
16byte
(패딩 비트로 16byte)


struct data
{
	char a;
	short c;
	int b;
	float d;
};
12byte
(패딩 비트가 줄어 4byte 줄임)
