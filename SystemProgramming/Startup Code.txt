###Startup 코드

- C프로그램의 실행을 위한 준비과정을 처리하는 코드
- 실행 시점에서 여전히 시스템에서는 C코드를 인식할 수 없어 어셈블리로 작성함
- 부트 코드와, 스타트업 코드를 하나의 파일로 만들어서 사용

cf. 부트 코드 : H/W 초기화로, 클럭 설정, 메모리와 H/W 무결성 체크 등



## 절차

1. 인터럽트 사용불가(disable)로 설정

스타트업 코드가 완료되기 전에 인터럽트가 발생하면, 아직 핸들러가 준비되지 않은 메모리로 분기해 제어 불능


2. RAM으로 rw_data 복사

3. ZI(bss) 영역 0으로 클리어

4. 모드별 stack, heap을 생성

5. 인터럽트 사용가능(enable)로 설정

6. main() 호출
