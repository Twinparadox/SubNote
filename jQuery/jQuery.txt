### jQuery

- 브라우저에서 동작하는 Client Side JavaScript Library
- JavaScript 프로그래밍을 단순화 -> 간결한 코드로 많은 동작 구현
- 문서 객체 모델 조작, 이벤트 제어, 애니메이션, AJAX 개발
- 웹 표준만으로 플래시와, 실버라이트 사용과 유사한 효과 구현
- 모바일에서도 작동 가능
- 브라우저 간 차이로 발생하는 문제점 쉽게 해결 가능

* CDN(Content Delivery Network) : 콘텐츠를 효율적으로 전달하기 위해 여러 노드를 가진 네트워크에서 데이터를 저장하여 제공하는 방식
- 분산된 서버를 통해 jQuery를 사용할 수 잇게 하여 성능 향상
- 로딩 속도 향상(다운된 jQuery의 복사본이 있기 때문)
- CDN 호스트 : MS, Google



## 전체 선택자
- HTML 문서 내 모든 태그 선택
- 선택자는 문자열로 취급하므로 '', ""로 감싸줌


## 태그 선택자
- 태그 이름에 근거해 태그를 선택하며 이용
- 여러 태그 선택 시, 콤마로 구분
- css() 메서드 : 선택된 태그 스타일 속성 설정, 리턴 시 사용, 설정하고자 하는 속성과 값을 매개 변수로 입력, 여러 속성 변경할 경우 콜론으로 속성과 값을 연결하고 콤마로 구분한 것을 중괄호{}로 묶음


## 아이디 선택자
- HTML 태그의 id 속성 이용
- # 다음 id 지정
- 하나의 특정 태그 선택을 위해 문서 내 id 중복이 없어야 함


## 클래스 선택자
- 특정 클래스를 가지는 태그 찾을 때 사용
- (.)를 이용해 선택
- 클래스 속성을 모두 갖는 태그를 선택하려면 클래스 선택자를 붙여서 사용
